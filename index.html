Tree of Life Project

The “Tree of Life” project is a versatile and responsive company site developed as a part of my junior development journey. It is structured with a sample landing page and a product page, both showcasing the diligent application of flexbox technology, ensuring the site’s aesthetic appeal and functionality across diverse devices The development journey involved the implementation of HTML5, CSS3, JavaScript, and Flexbox, focusing on crafting a refined and user-friendly experience. The interactive dynamics powered by JavaScript include on-hover effects, click-based interactions, real-time data fetching for product information, and a captivating image slider animation.
Feel free to browse through the project's GitHub Repository

Book Club Project

The Book Club project is an interactive and responsive website designed as a unique platform for book enthusiasts. The website is implemented using HTML5, CSS3, Flexbox, and JavaScript, creating a user-friendly and aesthetically pleasing interface that adapts well across a variety of devices. The website features an events section with dynamic dropdowns for more information, a partner section showcasing collaborators, and an inviting contact form page for user inquiries and feedback, each enhancing the user interaction and experience on the site.
